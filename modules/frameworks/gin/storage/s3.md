# Gin + AWS S3

> Gin-specific patterns for S3 using `aws-sdk-go-v2`.
> Generic S3 patterns are in `modules/frameworks/go-stdlib/storage/s3.md`.

## Integration Setup

```go
// go.mod
require (
    github.com/gin-gonic/gin v1.10.0
    github.com/aws/aws-sdk-go-v2/service/s3 v1.60.0
    github.com/aws/aws-sdk-go-v2/feature/s3/manager v1.17.0
)
```

## File Upload Endpoint

```go
func UploadHandler(s3Client *s3.Client, bucket string) gin.HandlerFunc {
    uploader := manager.NewUploader(s3Client)
    return func(c *gin.Context) {
        // Limit upload size at the framework level
        c.Request.Body = http.MaxBytesReader(c.Writer, c.Request.Body, 50<<20) // 50MB

        file, header, err := c.Request.FormFile("file")
        if err != nil {
            c.JSON(http.StatusBadRequest, gin.H{"error": "file required"})
            return
        }
        defer file.Close()

        key := fmt.Sprintf("uploads/%s/%s", uuid.NewString(), header.Filename)
        _, err = uploader.Upload(c.Request.Context(), &s3.PutObjectInput{
            Bucket:      aws.String(bucket),
            Key:         aws.String(key),
            Body:        file,
            ContentType: aws.String(header.Header.Get("Content-Type")),
        })
        if err != nil {
            c.JSON(http.StatusInternalServerError, gin.H{"error": "upload failed"})
            return
        }

        c.JSON(http.StatusCreated, gin.H{"key": key})
    }
}
```

## Presigned URL Handler

```go
func PresignUploadHandler(s3Client *s3.Client, bucket string) gin.HandlerFunc {
    presigner := s3.NewPresignClient(s3Client)
    return func(c *gin.Context) {
        key := fmt.Sprintf("uploads/%s/%s", uuid.NewString(), c.Query("filename"))

        req, err := presigner.PresignPutObject(c.Request.Context(), &s3.PutObjectInput{
            Bucket: aws.String(bucket),
            Key:    aws.String(key),
        }, s3.WithPresignExpires(15*time.Minute))
        if err != nil {
            c.JSON(http.StatusInternalServerError, gin.H{"error": "presign failed"})
            return
        }

        c.JSON(http.StatusOK, gin.H{
            "upload_url": req.URL,
            "key":        key,
            "expires_in": 900,
        })
    }
}
```

## Additional Dos/Don'ts

- DO use `http.MaxBytesReader` before `c.Request.FormFile` to reject oversized uploads at the connection level
- DO generate a UUID prefix for S3 keys to avoid filename collisions and directory hot-spots
- DO prefer presigned URLs for large files — avoids proxying binary data through your API
- DO validate `Content-Type` of uploaded files against an allowlist before storing
- DON'T store the raw user-supplied filename as the S3 key — sanitize or replace it with a UUID
- DON'T return the full S3 URL to clients — return the key and generate presigned download URLs on demand
