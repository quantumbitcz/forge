# ASP.NET Core + AWS S3 (AWSSDK.S3)

> ASP.NET Core-specific patterns for object storage via AWSSDK.S3. Extends generic ASP.NET conventions.

## Integration Setup

```bash
dotnet add package AWSSDK.S3
dotnet add package AWSSDK.Extensions.NETCore.Setup
```

```csharp
// Program.cs
builder.Services.AddDefaultAWSOptions(builder.Configuration.GetAWSOptions());
builder.Services.AddAWSService<IAmazonS3>();
// appsettings.json: "AWS": { "Region": "eu-central-1" }
// Credentials: IAM role (ECS/EKS) or environment: AWS_ACCESS_KEY_ID + AWS_SECRET_ACCESS_KEY
```

```csharp
// appsettings.json
{
  "AWS": { "Region": "eu-central-1" },
  "S3": {
    "Bucket": "my-app-bucket",
    "PresignedUrlTtlMinutes": 15
  }
}
```

## Framework-Specific Patterns

### Storage service with IAmazonS3 DI

```csharp
// Application/Services/StorageService.cs
public class StorageService(IAmazonS3 s3,
                            IConfiguration config,
                            ILogger<StorageService> logger)
{
    private readonly string _bucket = config["S3:Bucket"]!;
    private readonly int _ttl = int.Parse(config["S3:PresignedUrlTtlMinutes"] ?? "15");

    public async Task<string> UploadAsync(string key, Stream content, string contentType,
                                          CancellationToken ct = default)
    {
        var request = new PutObjectRequest
        {
            BucketName  = _bucket,
            Key         = key,
            InputStream = content,
            ContentType = contentType,
            AutoCloseStream = false,
        };
        await s3.PutObjectAsync(request, ct);
        return key;
    }

    public string GetPresignedDownloadUrl(string key)
    {
        var request = new GetPreSignedUrlRequest
        {
            BucketName = _bucket,
            Key        = key,
            Verb       = HttpVerb.GET,
            Expires    = DateTime.UtcNow.AddMinutes(_ttl),
        };
        return s3.GetPreSignedURL(request);
    }

    public string GetPresignedUploadUrl(string key, string contentType)
    {
        var request = new GetPreSignedUrlRequest
        {
            BucketName  = _bucket,
            Key         = key,
            Verb        = HttpVerb.PUT,
            ContentType = contentType,
            Expires     = DateTime.UtcNow.AddMinutes(5),
        };
        return s3.GetPreSignedURL(request);
    }
}
```

### TransferUtility for large files

```csharp
using var transferUtility = new TransferUtility(s3);
await transferUtility.UploadAsync(new TransferUtilityUploadRequest
{
    BucketName   = _bucket,
    Key          = key,
    FilePath     = localPath,
    PartSize     = 6 * 1024 * 1024,   // 6 MB parts
    StorageClass = S3StorageClass.IntelligentTiering,
});
```

## Scaffolder Patterns

```
Application/
  Services/
    StorageService.cs           # upload, presigned URL, delete
    IStorageService.cs          # interface for testability
Infrastructure/
  DependencyInjection.cs        # AddAWSService<IAmazonS3>
```

## Dos

- Register `IAmazonS3` via `AddAWSService<IAmazonS3>()` so AWS SDK manages connection pooling and credentials refresh
- Use `TransferUtility.UploadAsync` for files > 5 MB — it handles multipart automatically
- Use presigned URLs for client-initiated uploads/downloads — avoid streaming through the ASP.NET process
- Inject `IAmazonS3` via constructor DI so services are unit-testable with `AmazonS3Client` mocks

## Don'ts

- Don't instantiate `AmazonS3Client` directly with hardcoded credentials — use `DefaultAWSCredentials` / IAM
- Don't call `GetPreSignedURL` (synchronous) from a hot async path — it is CPU-only but blocks thread pool briefly; batch presign URLs offline
- Don't use `PutObjectRequest.FilePath` for large files in web requests — stream from `IFormFile.OpenReadStream()`
- Don't skip `AutoCloseStream = false` when passing a stream you control the lifetime of
