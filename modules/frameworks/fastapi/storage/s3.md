# FastAPI + AWS S3 (aioboto3)

> FastAPI-specific patterns for object storage via aioboto3 (async boto3 wrapper). Extends generic FastAPI conventions.

## Integration Setup

```bash
aioboto3==13.1.0
```

```python
# app/config.py (extends existing Settings)
class Settings(BaseSettings):
    aws_region: str = "eu-central-1"
    aws_s3_bucket: str
    aws_endpoint_url: str | None = None     # LocalStack override
    aws_presigned_url_ttl: int = 900        # seconds
```

```python
# app/s3_client.py
import aioboto3
from app.config import settings

session = aioboto3.Session()

async def get_s3():
    async with session.client(
        "s3",
        region_name=settings.aws_region,
        endpoint_url=settings.aws_endpoint_url,
    ) as client:
        yield client
```

## Framework-Specific Patterns

### Dependency type alias

```python
from typing import Annotated
from fastapi import Depends

S3Dep = Annotated[Any, Depends(get_s3)]
```

### Async upload endpoint

```python
from fastapi import UploadFile, HTTPException
import uuid

@router.post("/files/upload", status_code=201)
async def upload_file(file: UploadFile, s3: S3Dep) -> dict:
    key = f"uploads/{uuid.uuid4()}/{file.filename}"
    try:
        await s3.upload_fileobj(
            file.file,
            settings.aws_s3_bucket,
            key,
            ExtraArgs={"ContentType": file.content_type or "application/octet-stream"},
        )
    except Exception as exc:
        raise HTTPException(500, detail="Upload failed") from exc
    return {"key": key}
```

### Presigned URL generation

```python
@router.get("/files/{key:path}/url")
async def presigned_url(key: str, s3: S3Dep) -> dict:
    url = await s3.generate_presigned_url(
        "get_object",
        Params={"Bucket": settings.aws_s3_bucket, "Key": key},
        ExpiresIn=settings.aws_presigned_url_ttl,
    )
    return {"url": url}
```

### Presigned upload (client-side direct PUT)

```python
@router.post("/files/presign-upload")
async def presign_upload(filename: str, content_type: str, s3: S3Dep) -> dict:
    key = f"uploads/{uuid.uuid4()}/{filename}"
    url = await s3.generate_presigned_url(
        "put_object",
        Params={"Bucket": settings.aws_s3_bucket, "Key": key, "ContentType": content_type},
        ExpiresIn=300,
    )
    return {"upload_url": url, "key": key}
```

## Additional Dos/Don'ts

- DO yield the S3 client inside `async with` — aioboto3 context managers handle connection lifecycle
- DO use `generate_presigned_url` for client-side uploads/downloads — never stream large blobs through FastAPI
- DO set `ContentType` via `ExtraArgs` on upload — S3 default is `binary/octet-stream`
- DON'T store the `aioboto3.Session` per-request — create it once at module level and reuse
- DON'T catch bare `Exception` in production; catch `botocore.exceptions.ClientError` and inspect `Error.Code`
- DON'T hardcode bucket or region — read from `Settings` so tests can override with LocalStack
