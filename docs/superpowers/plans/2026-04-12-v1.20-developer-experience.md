> **For agentic workers:** REQUIRED SUB-SKILL: Use superpowers:subagent-driven-development (recommended) or superpowers:executing-plans to implement this plan task-by-task. Steps use checkbox (`- [ ]`) syntax for tracking.

**Goal:** Complete the forge enhancement roadmap with developer experience features: auto-wiki, autonomous memory discovery, pipeline timeline, codebase Q&A, and insights dashboard.

**Architecture:** Five features. Wiki generator (Feature 17) is the only new agent (fg-135). Memory discovery (Feature 18) extends the retrospective. Timeline (Feature 19) extends post-run. Q&A and insights (Features 20-21) are new skills. All independent — no cross-dependencies.

**Tech Stack:** Markdown (agents, contracts, skills), JSON (state schema, registry), BATS (contract tests)

**Implementation Order:** All features are independent — batch for maximum parallelism.
