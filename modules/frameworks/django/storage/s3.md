# Django + S3 (django-storages)

> File and static asset storage on S3 using `django-storages[s3]` and `FileField`.

## Integration Setup

```bash
pip install "django-storages[s3]" boto3
```

```python
# settings.py
INSTALLED_APPS = [..., "storages"]

AWS_ACCESS_KEY_ID     = env("AWS_ACCESS_KEY_ID")
AWS_SECRET_ACCESS_KEY = env("AWS_SECRET_ACCESS_KEY")
AWS_STORAGE_BUCKET_NAME = env("S3_BUCKET")
AWS_S3_REGION_NAME    = env("AWS_REGION", default="eu-central-1")
AWS_S3_FILE_OVERWRITE = False        # never overwrite uploaded files
AWS_DEFAULT_ACL       = None         # use bucket policy, not per-object ACLs
AWS_QUERYSTRING_AUTH  = True         # presigned URLs (private bucket)
AWS_S3_CUSTOM_DOMAIN  = env("CDN_DOMAIN", default=None)  # optional CloudFront
```

## Framework-Specific Patterns

### Default and static storage
```python
# settings.py
STORAGES = {
    "default": {"BACKEND": "storages.backends.s3boto3.S3Boto3Storage"},
    "staticfiles": {"BACKEND": "storages.backends.s3boto3.S3StaticStorage"},
}
```

### Custom storage backend (separate media vs. static bucket)
```python
# storage_backends.py
from storages.backends.s3boto3 import S3Boto3Storage

class PublicStaticStorage(S3Boto3Storage):
    bucket_name = settings.STATIC_BUCKET
    default_acl = "public-read"
    querystring_auth = False

class PrivateMediaStorage(S3Boto3Storage):
    bucket_name = settings.MEDIA_BUCKET
    default_acl = None
    querystring_auth = True
    file_overwrite = False
```

### FileField with S3 storage
```python
# products/models.py
from storage_backends import PrivateMediaStorage

class Product(models.Model):
    name = models.CharField(max_length=200)
    image = models.ImageField(storage=PrivateMediaStorage(), upload_to="products/%Y/%m/")

    @property
    def image_url(self):
        return self.image.url  # generates presigned URL when querystring_auth=True
```

### Collecting static files
```bash
python manage.py collectstatic --noinput   # uploads to S3 static bucket
```

## Scaffolder Patterns
```
myapp/
  storage_backends.py   # PublicStaticStorage, PrivateMediaStorage
  models.py             # FileField / ImageField with storage=
  settings/
    base.py             # AWS_* vars from env
    production.py       # STORAGES override
```

## Dos
- Use separate buckets for static and media files — different ACL policies
- Set `AWS_S3_FILE_OVERWRITE = False` to prevent name collisions on upload
- Use `upload_to` callable or date-based path to avoid bucket root flooding
- Use environment variables for all AWS credentials — never hardcode

## Don'ts
- Don't set `AWS_DEFAULT_ACL = "public-read"` for private uploads — use presigned URLs
- Don't call `collectstatic` with `--clear` in CI unless the bucket is versioned
- Don't use `DEFAULT_FILE_STORAGE` (legacy string) — use the `STORAGES` dict (Django 4.2+)
- Don't store `AWS_SECRET_ACCESS_KEY` in version control
