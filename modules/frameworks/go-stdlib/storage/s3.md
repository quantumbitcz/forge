# Go stdlib + AWS S3 (aws-sdk-go-v2)

> Go stdlib S3 patterns using `aws-sdk-go-v2`. Extends generic Go conventions.

## Integration Setup

```go
// go.mod
require (
    github.com/aws/aws-sdk-go-v2 v1.30.0
    github.com/aws/aws-sdk-go-v2/config v1.27.0
    github.com/aws/aws-sdk-go-v2/service/s3 v1.60.0
    github.com/aws/aws-sdk-go-v2/service/s3/s3manager // included in aws-sdk-go-v2/service/s3
)
```

## S3 Client Setup

```go
import (
    "github.com/aws/aws-sdk-go-v2/aws"
    "github.com/aws/aws-sdk-go-v2/config"
    "github.com/aws/aws-sdk-go-v2/service/s3"
)

func NewS3Client(ctx context.Context) (*s3.Client, error) {
    cfg, err := config.LoadDefaultConfig(ctx,
        config.WithRegion("eu-central-1"),
    )
    if err != nil {
        return nil, fmt.Errorf("load aws config: %w", err)
    }
    return s3.NewFromConfig(cfg), nil
}
```

Credentials are resolved from environment (`AWS_ACCESS_KEY_ID`, `AWS_SECRET_ACCESS_KEY`), IAM instance profile, or ECS task role — never hardcode them.

## Presigned URLs

```go
func PresignUpload(ctx context.Context, client *s3.Client, bucket, key string, ttl time.Duration) (string, error) {
    presigner := s3.NewPresignClient(client)
    req, err := presigner.PresignPutObject(ctx, &s3.PutObjectInput{
        Bucket: aws.String(bucket),
        Key:    aws.String(key),
    }, s3.WithPresignExpires(ttl))
    if err != nil {
        return "", fmt.Errorf("presign put: %w", err)
    }
    return req.URL, nil
}

func PresignDownload(ctx context.Context, client *s3.Client, bucket, key string, ttl time.Duration) (string, error) {
    presigner := s3.NewPresignClient(client)
    req, err := presigner.PresignGetObject(ctx, &s3.GetObjectInput{
        Bucket: aws.String(bucket),
        Key:    aws.String(key),
    }, s3.WithPresignExpires(ttl))
    if err != nil {
        return "", fmt.Errorf("presign get: %w", err)
    }
    return req.URL, nil
}
```

## Multipart Upload

```go
func UploadMultipart(ctx context.Context, client *s3.Client, bucket, key string, r io.Reader, contentType string) error {
    uploader := manager.NewUploader(client, func(u *manager.Uploader) {
        u.PartSize = 10 * 1024 * 1024 // 10MB parts
        u.Concurrency = 3
    })
    _, err := uploader.Upload(ctx, &s3.PutObjectInput{
        Bucket:      aws.String(bucket),
        Key:         aws.String(key),
        Body:        r,
        ContentType: aws.String(contentType),
    })
    return err
}
```

## Additional Dos/Don'ts

- DO use IAM roles (instance profile, ECS task role) for credentials in deployed environments — never static keys in config files
- DO always set a `ContentType` on uploads to enable correct browser handling on download
- DO use presigned URLs to grant temporary direct client access rather than proxying file bytes through the API
- DO set a TTL on presigned URLs appropriate to your use case (e.g., 15 minutes for uploads, 1 hour for downloads)
- DON'T call `GetObject` and stream large files through your application server — use presigned download URLs instead
- DON'T store bucket names as hardcoded strings — inject them via environment variables
- DON'T ignore the error from `uploader.Upload` — multipart failures can leave incomplete uploads consuming storage
